## 숫자 야구 게임

---
### 구현할 기능 목록

- 3자리 난수 생성
  - 1에서 9까지 서로 다른 임의의 수 3가지


- 사용자 숫자 입력받기
  - 입력값 숫자 3자리인지 검증


- 사용자가 입력한 숫자와 컴퓨터의 숫자 비교해서 결과 출력
  - 같은 수가 같은 자리에 있을 때 : 스트라이크
  - 같은 수가 다른 자리에 있을 때 : 볼
  - 같은 수가 전혀 없을 때 : 낫싱
  - 3가지 숫자 모두 같은 자리에 같은 수일 때 : 플레이어 승리


- 사용자 숫자 입력 및 결과 출력 과정을 플레이어가 승리할 때까지 반복
  - 게임 종료 시 게임을 다시 시작하거나 완전히 종료 중 선택값 입력받기
